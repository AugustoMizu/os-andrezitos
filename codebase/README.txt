Nome do jogo: cArImbo

Nome da equipe: Os Andrézitos 

Classificação: LIVRE

Descrição:
	O JOGO tem o objetivo de analisar documentos, dicidindo se são úteis ou não, de personagens que se voluntariaram para trabalhar em uma empresa que está criando uma IA. Para então carimba-lo como aprovado ou reprovado, com base no conteúdo.
Arcaico.


Contoles: 

- mouse esquerdo: interagir
- mouse direito: arrastar
- tecla P: pausar e sair
